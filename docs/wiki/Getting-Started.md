# 시작하기

**5분 안에 Git Tab을 설치하고 첫 저장소를 열어보세요.**

---

## 1단계: 설치하기

### 다운로드
1. [GitHub Releases 페이지](https://github.com/BaeTab/GitTab/releases)로 이동합니다.
2. 최신 버전의 **`GitTab-Setup.exe`** (또는 `*-installer.exe`) 파일을 찾아 다운로드합니다.
3. 다운로드한 파일을 두 번 클릭합니다.

### 설치 진행
인스톨러가 시작되면:

- **언어 선택:** 원하는 언어를 선택합니다 (기본값: 영어, Git Tab 실행 후 한국어로 변경 가능).
- **.NET 8 런타임 확인:** 처음 설치할 때 `.NET 8 Desktop Runtime`이 필요하다고 안내될 수 있습니다. "설치" 또는 "다운로드"를 클릭하면 자동으로 다운로드·설치됩니다. **(Git은 이미 포함되어 있어 별도 설치 불필요합니다.)**
- **설치 폴더:** 기본 폴더(`C:\Program Files\GitTab` 등)를 선택하고 계속 진행합니다.
- **완료:** 인스톨러가 완료되면 "Git Tab 실행" 체크박스가 활성화됩니다. 체크 후 "마침"을 클릭하면 앱이 바로 실행됩니다.

---

## 2단계: 첫 실행

### 앱 실행 확인
Git Tab 창이 열리면 성공입니다! 

- 상단 메뉴와 중앙 버튼들이 보입니다.
- 좌측에 "최근 저장소" 목록이 있습니다 (처음엔 비어있음).

### 한국어로 변환 (선택)
영어로 보이는 경우:
1. 우상단 **⚙ 설정** 아이콘을 클릭합니다.
2. **언어(Language)** 드롭다운에서 **한국어**를 선택합니다.
3. 앱이 자동 재시작되고 한국어로 변환됩니다.

---

## 3단계: 저장소 열기

### 방법 0: 원격 저장소 복제 (Clone)
GitHub, GitLab 같은 곳에 있는 저장소를 처음 받고 싶다면:
1. Git Tab 메인 창의 **"복제(Clone)"** 버튼을 클릭합니다.
2. 저장소 URL을 입력합니다 (예: `https://github.com/user/repo.git`).
3. 저장할 폴더를 선택하고 "복제" 버튼을 클릭하면 전체 저장소가 내려받아집니다.

### 방법 1: 버튼으로 열기
1. Git Tab 메인 창의 **"폴더 열기"** 또는 **"저장소 열기"** 버튼을 클릭합니다.
2. 기존 Git 저장소 폴더(`.git` 폴더가 있는 폴더)를 선택합니다.
3. **열기** 버튼을 클릭하면 저장소가 로드됩니다.

> **팁:** 아직 Git 저장소가 없다면 일반 폴더를 열고, Git Tab 메뉴에서 "새 저장소 초기화"를 선택해 만들 수 있습니다.

### 방법 2: 드래그 & 드롭
1. Windows 탐색기에서 폴더를 찾습니다.
2. 그 폴더를 Git Tab 창으로 끌어다 놓습니다.
3. 저장소가 자동으로 로드됩니다.

### 방법 3: Windows 탐색기 우클릭
1. 탐색기에서 폴더를 우클릭합니다.
2. 메뉴에서 **"Git Tab"** 또는 **"열기"**를 선택합니다.
3. Git Tab이 열리고 해당 저장소가 로드됩니다.

> **참고:** 우클릭 메뉴가 보이지 않으면 Git Tab 설정에서 "Windows 탐색기 우클릭 메뉴" 옵션을 켜야 합니다. ([탐색기 우클릭 페이지](탐색기-우클릭) 참고)

## 저장소(Repository)가 뭔가요?

**저장소 = 프로젝트 폴더 + Git 기록**

Git 저장소는 단순히 `.git` 폴더가 포함된 폴더입니다. 이 폴더 안에는 모든 파일 변경 이력이 저장됩니다. 덕분에 언제든 과거 상태로 돌아가거나 다른 사람과 협업할 수 있습니다.

- **다른 문제:** [자주 묻는 질문](자주-묻는-질문)에서 답을 찾아보세요.
